# cljs-ws

A Leiningen template for creating new clojurescript projects that use httpkit for websockets communication!

## Usage

    % lein new cljs-ws orb
    % cd orb
    % lein cljsbuild once
    % lein run
    
Clojurescript is communicating to HTTPKit over websockets!
